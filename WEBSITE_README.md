# 惜池移民官网 - 网站说明

## 项目概述

这是为"惜池移民"（Xichi Immigration）公司开发的官方网站。惜池移民是一家专业的英国移民服务公司，提供包括创业移民、全球人才签证（GTV）、技术工作签证等在内的全方位英国移民服务。

## 主要功能

### 1. 首页展示
- **公司品牌展示**：展示惜池移民的品牌形象和服务理念
- **服务介绍**：详细介绍四大核心服务
  - 创业移民（Startup Visa）
  - 全球人才签证（GTV）
  - 技术工作签证（Skilled Worker Visa）
  - 其他签证服务
- **公司介绍**：展示公司价值观和优势
- **行动号召**：引导用户进行GTV评估或AI咨询

### 2. GTV资格评估
- **AI智能评估**：用户上传简历或填写表单，AI自动分析GTV资格
- **详细报告**：提供详细的资格评分和改进建议
- **多领域支持**：支持数字技术、艺术文化、研究学术等领域

### 3. AI智能咨询
- **24/7在线咨询**：AI助手随时解答移民相关问题
- **个性化建议**：根据用户背景提供定制化建议
- **实时评估**：在咨询过程中实时评估用户资格

### 4. 关于我们页面
- **公司介绍**：详细介绍公司背景和优势
- **成功案例统计**：展示公司成功案例和成功率
- **价值观展示**：展示公司核心价值观
- **联系方式**：提供公司联系信息

## 技术特点

### 前端技术
- **Next.js 15**：使用最新的Next.js框架
- **React 19**：使用最新的React版本
- **TypeScript**：类型安全的开发
- **Tailwind CSS**：现代化的样式系统
- **shadcn/ui**：高质量的UI组件库

### 设计特点
- **响应式设计**：完美适配桌面、平板和手机
- **现代化UI**：精美的渐变效果和动画
- **双语支持**：支持中文和英文切换
- **深色模式支持**：支持深色/浅色主题切换

## 组件说明

### 核心组件
1. **Navbar** (`components/navbar.tsx`)
   - 网站导航栏，包含logo和主要菜单项
   - 支持移动端响应式菜单

2. **Hero** (`components/hero.tsx`)
   - 首页主横幅区域
   - 展示公司品牌和核心价值主张

3. **ServicesSection** (`components/services-section.tsx`)
   - 服务介绍区域
   - 展示四大核心服务及其特点

4. **AboutSection** (`components/about-section.tsx`)
   - 公司介绍区域
   - 展示公司价值观和优势

5. **Footer** (`components/footer.tsx`)
   - 网站页脚
   - 包含公司信息、服务链接和联系方式

## 使用说明

### 开发环境设置

1. 安装依赖：
```bash
npm install
# 或
pnpm install
```

2. 启动开发服务器：
```bash
npm run dev
# 或
pnpm dev
```

3. 访问网站：
打开浏览器访问 `http://localhost:3000`

### 生产环境部署

1. 构建项目：
```bash
npm run build
```

2. 启动生产服务器：
```bash
npm start
```

## 自定义配置

### 更新公司信息
- 修改 `components/footer.tsx` 中的联系信息
- 修改 `app/about/page.tsx` 中的公司介绍
- 更新 `app/layout.tsx` 中的网站元数据

### 添加新服务
1. 在 `components/services-section.tsx` 中添加新服务项
2. 创建对应的服务详情页面（如 `/services/[service-name]`）

### 修改样式
- 主要样式文件：`app/globals.css`
- 组件样式：使用 Tailwind CSS 类名
- 主题颜色：在 `app/globals.css` 中修改 CSS 变量

## 图片资源

网站需要以下图片资源（建议添加到 `public/` 目录）：

1. **公司Logo**：建议尺寸 200x200px，格式 PNG/SVG
2. **服务图标**：已使用 Lucide React 图标库，如需自定义图标可替换
3. **背景图片**：当前使用渐变背景，如需添加背景图片可修改相应组件

## 注意事项

1. **联系信息**：请更新 Footer 和 About 页面中的实际联系信息
2. **邮箱和电话**：当前为示例信息，需要替换为真实联系方式
3. **服务链接**：部分服务详情页面需要创建（如 `/services/startup`）
4. **AI功能**：AI咨询功能需要后端API支持，确保后端服务正常运行
