# GTV 申请策略规划

基于候选人评估结果，生成个性化的申请策略、时间规划、材料清单和改进建议。

## 策略规划框架

### 路径选择矩阵

| 条件 | 推荐路径 | 理由 |
|------|---------|------|
| 评分 80+，5年+ 行业经验 | Exceptional Talent | 已取得杰出成就 |
| 评分 70-80，3年+ 行业经验 | Exceptional Promise | 展现杰出潜力 |
| 评分 60-70，有创业想法 | Startup Visa → GTV | 先创业签证再转 GTV |
| 评分 < 60 | 提升后再申请 | 需要积累更多证据 |

### 申请时间规划

**标准时间线（6-9个月）**：

| 阶段 | 时长 | 任务 |
|------|------|------|
| 评估与定位 | 1-2周 | 评估资格、选择路径和背书机构 |
| 证据收集 | 1-3个月 | 收集并整理所有证据材料 |
| 推荐信获取 | 1-2个月 | 联系推荐人、撰写和修改推荐信 |
| 文案撰写 | 2-4周 | 个人陈述、证据描述、申请信 |
| 材料审核 | 1-2周 | 全面审查、翻译、公证 |
| 背书申请 | 提交后等 | 背书机构审核（通常5-8周） |
| 签证申请 | 背书后 | 获得背书后提交签证申请 |

**加急时间线（3-4个月）**：适用于材料基本就绪的申请人。

## 材料清单

### 必需材料

- [ ] 个人陈述 (Personal Statement)
- [ ] 3封推荐信 (Recommendation Letters)
- [ ] 简历/CV
- [ ] 护照复印件
- [ ] 证据材料包 (Evidence Bundle)

### 证据材料（按 MC/OC 组织）

**Mandatory Criteria 证据**：
- [ ] 学历证书和成绩单
- [ ] 工作证明/在职证明
- [ ] 项目文档/技术架构说明
- [ ] 薪资证明（证明行业地位）

**Optional Criteria 证据**（至少选2个维度）：
- [ ] 专利证书/申请文件
- [ ] 论文发表记录
- [ ] 会议演讲邀请函/议程
- [ ] 开源项目贡献截图
- [ ] 获奖证书
- [ ] 媒体报道
- [ ] 行业认证

## 改进建议生成

根据评估差距，按优先级生成建议：

**高优先级**（直接提升 MC 评分）：
- 获取知名人士推荐信
- 整理最具影响力的项目证据
- 争取高级职位变更

**中优先级**（提升 OC 评分）：
- 参与国际会议演讲
- 发表技术博客/论文
- 参与开源项目

**长期投资**（3-6个月见效）：
- 获取专业认证（AWS/GCP/PMP等）
- 建立技术社区影响力
- 申请专利

## 成功率预估

基于历史数据和评分的大致参考：

| 评分区间 | 预估通过率 | 建议 |
|---------|-----------|------|
| 85+ | 80-90% | 直接申请 |
| 75-84 | 60-75% | 补强弱项后申请 |
| 65-74 | 40-55% | 需要3-6个月准备 |
| < 65 | < 30% | 需要系统性提升 |
