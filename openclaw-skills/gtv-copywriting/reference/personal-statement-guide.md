# GTV 个人陈述写作指南

## Tech Nation 背书的个人陈述结构要求

个人陈述（Personal Statement）是 GTV 数字技术路径（Digital Technology）申请的核心材料之一，需经 Tech Nation 背书。其结构应清晰展示申请人在数字技术领域的成就、创新贡献和来英计划。

### 标准五段结构

| 段落 | 内容 | 评审对应 |
|------|------|----------|
| 1. 引言 | 自我介绍、申请动机、核心定位 | 建立第一印象 |
| 2. 专业背景与核心成就 | 教育、经历、关键项目、领导角色 | MC1-MC4 |
| 3. 创新贡献与行业影响 | 技术创新、行业认可、社会影响 | OC1-OC4 |
| 4. 来英计划 | 英国发展计划、贡献预期、时间线 | 背书机构关注 |
| 5. 总结 | 重申符合标准、简洁收尾 | 强化结论 |

## 各段落的字数预算

**总篇幅**：2000-3500 字（中英文混合时按有效字数计）

| 段落 | 字数预算 | 精炼原则 |
|------|----------|----------|
| 引言 | 200-300 字 | 开门见山，2-3 句建立身份和动机 |
| 专业背景与核心成就 | 800-1200 字 | 信的重心，充分使用数据和事实，覆盖 MC 四维 |
| 创新贡献与行业影响 | 600-800 字 | 突出 OC 维度，选最有分量的 2-3 项展开 |
| 来英计划 | 400-600 字 | 具体可行，与专业背景自然衔接 |
| 总结 | 100-200 字 | 重申核心判断，简洁有力 |

### 篇幅控制说明

- **低于 2000 字**：内容可能不够充实，需补充具体成就和数据
- **超出 3500 字**：必须精简，删除冗余修饰和重复论证
- **单段超预算**：优先压缩引言和总结，保留主体段落的论证深度

## 写作禁忌清单

### 空洞形容词（禁止使用）

| 禁止词 | 替代方式 |
|--------|----------|
| 令人瞩目的、极为突出的、尤为令人敬佩 | 用具体数据：如「带领 10 人团队，将系统性能提升 200%」 |
| 卓越的、杰出的、优秀的 | 用可验证的事实：如「获得 XX 奖项」「被 XX 媒体报道」 |
| 显著的、巨大的、重要的 | 用量化指标：如「用户增长 50 万」「年收入增长 120%」 |

### 主观判断（需事实支撑）

- ❌ 「我是该领域的领导者」→ ✅ 「担任 XX 公司 CTO，负责 50 人技术团队，主导 3 个千万级项目」
- ❌ 「我的贡献得到了广泛认可」→ ✅ 「获得 XX 奖、被 XX 日报报道、受邀在 XX 大会演讲」
- ❌ 「我的技术具有创新性」→ ✅ 「申请 2 项发明专利，论文发表于 XX 会议」

### 未支撑的主张

- 每个论点必须有具体事实或数据支撑
- 避免泛泛而谈，如「在多个项目中发挥关键作用」需指明项目名称和具体角色
- 避免模糊时间线，如「近年来」应改为「2022-2024 年」

## 优秀个人陈述的评判标准

| 维度 | 标准 |
|------|------|
| 结构完整性 | 五段结构清晰，逻辑递进 |
| 证据充分性 | 每段有 2-3 个具体事实或数据支撑 |
| MC/OC 覆盖度 | 至少覆盖 MC 四维，OC 至少 2 项 |
| 数据驱动 | 量化指标多于定性描述 |
| 语气得体 | 自信但不傲慢，专业且真实 |
| 篇幅合规 | 总字数 2000-3500，各段在预算内 |

## 不同路径的侧重差异

### Exceptional Talent（杰出人才）

- **侧重**：已取得的成就、行业领导地位、国际影响力
- **语气**：更强调「已证明」「已建立」「已获得」
- **证据选择**：优先使用最能体现「公认领导者」的证据
- **来英计划**：强调如何将已有影响力迁移至英国

### Exceptional Promise（杰出潜力）

- **侧重**：早期成就、成长轨迹、潜力证明
- **语气**：可适当使用「正在建立」「展现出」「早期迹象」
- **证据选择**：优先使用最能体现「新兴领导者」的证据，如早期项目、快速成长数据
- **来英计划**：强调英国如何提供更好的发展环境

### Startup Visa（初创签证）

- **侧重**：商业计划、创新性、英国市场机会
- **证据选择**：商业验证、团队、融资、市场调研
- **来英计划**：与商业计划紧密结合，说明在英国落地的具体步骤
